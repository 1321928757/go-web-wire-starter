# go-captcha-example-vue

## Project setup
```
npm install
```
In the project directory, you can run:

### Compiles and hot-reloads for development
```
npm run serve
```

## Start backend
```
git clone github.com/wenlng/go-captcha-example
cd go-captcha-example
go mod download
go run main.go
```
